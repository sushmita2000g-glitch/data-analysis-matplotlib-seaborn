# 📊 Amazon Data Analysis Project (Matplotlib & Seaborn)

## 📌 Project Overview
This project focuses on analyzing an Amazon dataset using Python libraries such as Matplotlib and Seaborn. The main goal is to explore the dataset, uncover hidden patterns, and present insights through advanced data visualizations and dashboards.

---

## 🛠️ Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📂 Dataset
The dataset contains Amazon customer data, including purchase behavior, order values, and customer insights.

---

## 📈 Key Features of the Project

### 🔹 Data Processing
- Data cleaning and sorting
- Filtering based on different conditions
- Sorting by order values and counts

### 🔹 Visualizations Created
- Box Plot
- Violin Plot
- Strip Plot
- Swarm Plot
- Ridge Plot
- Histogram
- Bar Plot
- Horizontal Bar Plot
- Pie Chart
- Heatmap
- Count Plot
- Catplot

### 🔹 Dashboards
- 📊 Dynamic dashboard for customer data
- 📊 Dashboard with filters (e.g., total bill, order value)
- 📊 Visualization combining multiple charts in one image

### 🔹 Analysis Performed
- Top 10 customers by total order value
- Customer purchase patterns
- Order distribution analysis
- Category-based comparison
- Correlation analysis using heatmaps

---

## 📊 Key Insights
- Identified top-performing customers based on total spending
- Observed patterns in order frequency and value
- Found relationships between different variables using heatmaps
- Visualized distribution of orders using histograms

---

## 🚀 How to Run the Project
1. Download the repository
2. Open the notebook in Jupyter Notebook or Google Colab
